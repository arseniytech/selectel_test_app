# Тестовое задание Selectel

Простое приложение на Angular 19

## Как запустить

```
npm install
npm start
```

Откроется на localhost:4200

## Что сделано

- Главная страница с приветствием
- Страница меню с разделами
- Чекбоксы для выбора элементов  
- Счетчики выбранных элементов и их суммы

Сделано с помощью Claude AI
